# 🌍 실용 영어 학습 홈페이지

일상생활에서 가장 많이 사용하는 영어 표현들을 카테고리별로 정리하고 학습할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

### 📚 6개 학습 카테고리
- **👋 인사하기**: 기본적인 인사말과 대화 시작
- **🏠 일상생활**: 집, 학교, 직장에서 자주 사용하는 표현
- **🛒 쇼핑하기**: 상점, 레스토랑에서 사용하는 표현
- **✈️ 여행하기**: 여행, 교통수단 관련 표현
- **🚨 긴급상황**: 도움이 필요할 때 사용하는 표현
- **😊 감정표현**: 기쁨, 슬픔, 화남 등 감정 표현

### 🎯 학습 기능
- **총 48개의 실용적인 영어 문장** 제공
- 각 문장마다 한국어 번역과 사용법 설명
- 이전/다음 버튼으로 순차 학습
- 진행률 표시 (현재 문장 / 전체 문장 수)

### 💻 사용자 경험
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기 지원
- **키보드 단축키**: 화살표 키로 문장 이동, ESC로 뒤로가기
- **터치 제스처**: 모바일에서 스와이프로 문장 이동
- **로컬 스토리지**: 학습 진행률 자동 저장
- **다크 모드**: 시스템 설정에 따른 자동 테마 변경

## 🚀 시작하기

### 1. 파일 다운로드
프로젝트의 모든 파일을 다운로드합니다:
- `index.html` - 메인 HTML 파일
- `styles.css` - 스타일시트
- `script.js` - JavaScript 기능

### 2. 웹 브라우저에서 열기
`index.html` 파일을 더블클릭하거나 웹 브라우저에서 열기

### 3. 학습 시작
1. 원하는 카테고리 선택
2. 영어 문장, 한국어 번역, 사용법 확인
3. 이전/다음 버튼으로 문장 이동
4. 키보드 화살표 키나 모바일 스와이프로도 사용 가능

## 🎨 디자인 특징

- **모던한 UI**: 글래스모피즘(Glassmorphism) 디자인 적용
- **그라데이션 배경**: 시각적으로 매력적인 색상 구성
- **부드러운 애니메이션**: 호버 효과와 전환 애니메이션
- **직관적인 네비게이션**: 사용하기 쉬운 인터페이스

## 📱 지원 기기

- ✅ 데스크톱 (Windows, macOS, Linux)
- ✅ 태블릿 (iPad, Android 태블릿)
- ✅ 모바일 (iPhone, Android 스마트폰)
- ✅ 모든 최신 웹 브라우저

## 🔧 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션, 반응형 디자인
- **JavaScript (ES6+)**: 모듈화된 코드, 이벤트 처리
- **로컬 스토리지**: 학습 진행률 저장
- **웹 폰트**: Noto Sans KR, Inter 폰트 사용

## 📖 학습 방법

### 효과적인 학습을 위한 팁
1. **매일 조금씩**: 하루에 5-10개 문장만 학습
2. **소리 내어 읽기**: 문장을 소리 내어 읽으면 기억에 더 잘 남음
3. **실제 상황 연습**: 학습한 문장을 실제 상황에서 사용해보기
4. **반복 학습**: 잘 외워지지 않는 문장은 여러 번 반복하기

### 카테고리별 학습 순서 추천
1. **인사하기** → 기본적인 소통 시작
2. **일상생활** → 일상에서 자주 사용
3. **쇼핑하기** → 해외여행이나 외국인과의 소통
4. **여행하기** → 여행 상황에서 활용
5. **긴급상황** → 위급할 때 사용
6. **감정표현** → 감정을 표현할 때 사용

## 🌟 특징

- **실용성**: 실제 일상생활에서 바로 사용할 수 있는 표현들
- **접근성**: 한국어 사용자를 위한 친숙한 인터페이스
- **확장성**: 새로운 카테고리나 문장을 쉽게 추가 가능
- **오프라인 지원**: 인터넷 연결 없이도 사용 가능

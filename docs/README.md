# 프로젝트 문서

이 디렉토리는 모든 공용 API, 함수 및 구성 요소의 포괄적인 문서를 담습니다. 코드가 추가될 때 새로운 엔트리를 생성하고, 예제와 사용 지침을 함께 제공합니다.

- `docs/api/`: 공개 API 레퍼런스 (자동/수동 생성)
- `docs/guides/`: 개념 설명, 사용 가이드, 모범 사례
- `docs/templates/`: 신규 모듈/컴포넌트 문서 템플릿
- `examples/`: 실행 가능한 예제 코드와 스니펫

문서 기여 방법:
1) 새로운 공개 엔트리(함수, 클래스, 컴포넌트)를 추가할 때 `docs/templates`의 템플릿으로 문서를 작성합니다.
2) `docs/api` 아래에 적절한 하위 경로를 만들고 문서 파일을 추가합니다.
3) 예제가 있으면 `examples/`에 파일을 만들고 문서에서 링크합니다.
4) 루트 `README.md`에 주요 변경 사항을 요약하고, 관련 문서 링크를 추가합니다.

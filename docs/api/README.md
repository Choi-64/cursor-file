# 공개 API 인덱스

현재 저장소에는 공개된 API/함수/컴포넌트가 등록되어 있지 않습니다.

새로 추가할 때는 다음 절차를 따르세요:
1) `docs/templates`에서 적절한 템플릿을 선택합니다.
2) `docs/api/{카테고리}/{이름}.md` 경로에 문서를 생성합니다.
3) 관련 예제 코드를 `examples/{카테고리}/{이름}/`에 추가합니다.
4) 이 인덱스에 링크를 추가합니다.

예시:
- 유틸리티: `docs/api/utils/parseDate.md`
- 컴포넌트: `docs/api/components/Button.md`
